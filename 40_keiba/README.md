# 競馬 投資回収計画

「趣味」ではなく「投資」として、収支・回収率(ROI)・資金管理を数値で回す領域。

## ファイル
- **[ledger.md](ledger.md)** — 収支台帳（1ベット1行）。すべての起点。
- **[rules.md](rules.md)** — 資金管理・ベットルール（破ってはいけない規律）
- **[analysis.md](analysis.md)** — 券種別・条件別の振り返りと戦略メモ
- `races/` — 注目レースの事前検討メモ（1レース1ファイル）

## 基本指標
- **回収率 (ROI)** = 払戻合計 ÷ 投資合計 × 100（100%で収支トントン）
- **的中率** = 的中件数 ÷ ベット件数
- **収支** = 払戻合計 − 投資合計

## 運用ルール（要点 → 詳細は rules.md）
1. **月の投資上限を先に決める**（生活費と完全分離）
2. 1レースの上限、券種を事前に決めてから買う
3. 熱くなったら止める（負け追いは禁止）
4. すべて ledger.md に記録（記録しない賭けはしない）

> 「今月の回収率を出して」「7月の収支を集計して」と Claude に頼めば ledger から計算します。
