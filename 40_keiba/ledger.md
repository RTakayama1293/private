# 🐎 収支台帳 (Ledger)

> **1ベット1行**で記録する。すべての集計はここから。
> 投資・払戻の単位は「円」。負けたベットは払戻=0。
> 「今月分を集計して」と頼めば下の表から回収率・収支を計算します。

## 記録項目
- 日付 / 開催（例: 東京11R）/ レース名
- 券種（単勝/複勝/馬連/ワイド/馬単/三連複/三連単 など）
- 買い目 / 投資額 / オッズ / 払戻 / 収支 / メモ
